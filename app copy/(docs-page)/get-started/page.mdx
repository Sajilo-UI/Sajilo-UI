<h5 className='text-3xl my-4 font-semibold'>Installation</h5>

<div className='text-lg leading-relaxed'>
  To get started with Sajilo UI, you need to install <code>tailwindcss</code>.
  Follow the steps below to set it up in your project.
</div>

<div className='list-decimal list-inside my-5'>
  <div className='my-2'>

    Install Tailwind CSS via npm:
     ```bash
      npm install tailwindcss
       ```

  </div>
  <div className='my-2'>
    Create a <code>tailwind.config.js</code> file:
      ```bash
      npx tailwindcss init
```
  </div>
  <div className='my-2'>
    Add Tailwind to your CSS:
      ```css
        @tailwind base;
        @tailwind components;
        @tailwind utilities;
        ```
  </div>
  <div className='my-2'>
    Start using Tailwind's utility classes in your HTML:

```html
<div className="text-center">Hello World!</div>
```

  </div>
</div>

<div className='text-lg leading-relaxed'>
  With these steps, you will have Tailwind CSS set up in your project, and you
  can start using Sajilo UI components to build beautiful and responsive user
  interfaces.
</div>
